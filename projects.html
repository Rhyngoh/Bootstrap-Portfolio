Project #1: Map Your Life!

First group project utilizing two API's to create an application. Map Your Life uses Google Maps API and Foursquare API to gather data of local restaurants or attractions near a given address. Further expansion can be made to add real estate data, rental data, and neighborhood data to find the perfect location to move to.

My contribution to this project includes: majority of the back end, ajax calls and queries, firebase implementation, markers to show on the map after a search, adding all of the categories, and creating new divs for each of the search results.

Technologies used:

  • Ajax
  • Firebase
  • Google Maps API
  • Foursquare API
  • Googe geocoding
