Projects

Map Your Life!

First group project utilizing two API's to create an application. Map Your Life uses Google Maps API and Foursquare API to gather data of local restaurants or attractions near a given address. Further expansion can be made to add real estate data, rental data, and neighborhood data to find the perfect location to move to.

My contribution to this project includes: majority of the back end, ajax calls and queries, firebase implementation, markers to show on the map after a search, adding all of the categories, and creating new divs for each of the search results.

Technologies used:

  • Ajax
  • Firebase
  • Google Maps API
  • Foursquare API
  • Googe geocoding

Yum-Yum Recipes

Second group project to create a full-stack application using node, express, and mysql. Yum-Yum Recipes seeks to provide an easier and more portable interface for finding recipes and creating a shopping list. Yum-Yum Recipes brings you the simplest shopping list/recipe finder app on the market! You can search by ingredient and even post your own recipes to expand our database of recipes! Administrators can delete recipes and access the MySQL database. Original recipe data from Spoonacular API.

My contribution to this project includes: primarily working on the backend, delete ajax call, local storage implementation, MySQL database architect, and improving UX

Technologies used:

  • MySQL
  • Sequelize ORM
  • MasonryJS
  • Ajax
  • local storage implementation
  • Heroku deployment
  • jQuery
  • Materialize CSS Framework
